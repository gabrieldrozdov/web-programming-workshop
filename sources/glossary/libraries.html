A library is something that extends the functionality of a programming language beyond its basic featureset. For web programming, we usually encounter these in the form of JavaScript libraries.

Some examples of JavaScript libraries include:

  • p5.js for creative coding and image generation.
  • Matter.js for physics simulation.
  • Tone.js for audio synthesis and sampling.
  • Three.js for working with 3D models.

An older popular library was jQuery, which significantly expanded JavaScript’s features for general use. Over time, JavaScript natively added much of jQuery’s functionality, so the library is no longer recommended for use.
