4-week Workshop Curriculum!

What will we learn?
Overview

The 4-week workshop curriculum focuses on HTML/CSS fundamentals, which are covered from lessons 1 to 10. The short timeframe means that there is little time to get into JavaScript, but if the cohort is doing well then we’ll delve into it with lessons 11 to 15.

Outcomes

In this workshop, you’ll learn how to:

  • Establish a typographic hierarchy using HTML tags.
  • Style content using inline CSS and classes.
  • Introduce basic interactivity using hover and active states.
  • Animate content using keyframes.
  • Adapt designs using responsive media queries.
  • Structure content using flexbox and CSS grid.

What are our projects?
Research and Analysis

We’ll start each class by talking about the design of other websites and breaking down how they might have been created. You’ll sign up for a presentation slot, during which you’ll do your best to analyze the parts of a website of your choosing. This analysis should include the different types of content on the website (text, images, videos), the types of interaction on the website (hover, click, scroll), and the ways the website changes or responds (animations, mobile layouts).

Project 1: 20 Lines to Tell a Story

Tell a story with code! Using 20 lines (or fewer) of HTML and CSS, paint a picture of some scene or narrative.

Project 2: Three Short Stories

Craft a tale with code! Create at least 3 websites that tell an overarching narrative

How will I be graded?
Requirements and Expectations

This course is graded on a rubric. This means that your grade exactly corresponds to the work you complete as part of the class.

See “Assessment” below to calculate your own grade for the course.

You are also free to take this course pass/fail. If you elect for this grading method, you still need to submit all assignments to pass.

Participation and Attendance

Attendance is required for all weekly meetings. If you have an excuse for missing a class, you must let me know at least 24 hours in advance. An unexcused absence is a 12.5% reduction of your grade.

If you miss a class, you are also responsible for catching up on the class’s lessons and completing any relevant exercises. These materials will always be available online at the class website.
